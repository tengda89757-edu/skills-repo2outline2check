# 跨学科研究检查清单详细指南

## 执行前准备

### 确定研究类型

```
Step 1: 识别研究领域
□ AI+教育
□ AI+医学
□ AI+社科
□ 其他跨学科领域

Step 2: 识别研究设计
□ 随机对照试验 (RCT)
□ 准实验研究
□ 队列研究
□ 案例研究
□ 混合方法研究
□ 纵向研究
□ 横断面研究

Step 3: 识别人类被试情况
□ 有人类被试 → 需要IRB审批
□ 使用公开数据集 → 需要数据使用协议
□ 无人类被试 → 仅AI技术评估
```

## AI技术维度检查清单

### 1. 算法/模型描述

```
□ 形式化定义
  - 是否有数学符号定义？
  - 符号是否一致？
  - 是否提供符号表？

□ 算法原理
  - 是否说明算法原理？
  - 是否有理论依据？
  - 是否引用原始论文？

□ 伪代码/流程图
  - 是否提供伪代码？
  - 流程图是否清晰？
  - 步骤是否完整？

□ 模型架构
  - 是否说明模型架构？
  - 是否提供架构图？

```

## 执行流程

```
Step 1: 研究类型识别
├── 确定研究领域
```
